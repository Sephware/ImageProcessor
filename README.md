# Image Editing Application
This app enables you to edit images by using filters and transformations. This project is primaraly Python along with OpenCV for image processing, and PyQt5 as the desktop framekwork.
## Features for Images
- Adjust image clarity (blur and sharpen).
- Edge Detection (Sobel Filtering).
- Image Transformations such as rotation, cropping, and scaling.
- Save new images in multiple formats (*.png, *.jpg, *.webp, *tiff, etc.)
